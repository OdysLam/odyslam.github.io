Hey,

Apparently this is my third attempt to keep a decent and up-to-date blog, where I will talk about technology and making.

I hope to document my projects and write posts that will serve as tutorials so as the readers can experiment for themselves.

My main project is GLaDOS, a smart-home assistant that control various appliances in my apartment as well as the front-door. The system is controlled via a web-interface and a facebook boot implementation. It uses an array of libraries and is based on python.

The first post will be about Jekyll, a wonderful tool that creates simple static blog-aware webpages in seconds. The best part is that it can be done without knowing anything about web-programming.This blog uses that platform and is hosted on Github for free.

I imagine that I will also be talking for random stuff in between, as writing seems pretty fun.
