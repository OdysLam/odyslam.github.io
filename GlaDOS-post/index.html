Let’s get into detail

In this post I am going to explain thoroughly all the parts of the project so as you have an idea of what to come

GLaDOS it’s just the name I have given to my project since she is one of my favorite villains in one of the best series I have every played.It would be wise to avoid building the actual GLaDOS since it might cause severe pain,or even death.

Finally! What does it do? (°ロ°)☝

As I have said a thousand times it’s a smart-home project which aims at automating and facilitating a number of tasks, everything is controlled from a central web-page that is run locally on the Raspberry pi from a python server.

Tasks:

  • Lock/Unlock of both the buildings door and the appartment’s. The first is done with the help of a releay and the second with the help of a servo and some 3D printed gears.

  • Control of 3 RF remote suko switches, by broadcasting a specific binary code I can turn on/off each switch remotely.

  • Control of Boilder. A power relay that uses AC as control signal is connected to an RF remote suko switch, same function as above. Bonus points for a software timer to fix the “Ooooh SHIT I forgot to turn off the boiler again”.

*Remote turn on of my computer by magic (packet).

Software:

To create the Interface, I built a simple .html page using html-css-javascript. In javascript I used the Jquery and webiopi libraries aswell.

To complete the tasks in the script file I use a number of python libraries (or the python library version of programs), namely:

  1. Webiopi
  2. wakeonlan
  3. vncdotool
  4. Pi_switch

Which we will get into detail when we examine the python code later in the guide.
